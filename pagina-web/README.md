# 🌐 Página Web

Este projeto é uma página web pessoal desenvolvida como parte do portfólio profissional, publicada via **GitHub Pages**.

## 🔗 Acesse a página

👉 https://vechi123.github.io/vechi-portifolio/

## 📋 Sobre o Projeto

A página foi criada para apresentar meu perfil profissional de forma online, contendo informações sobre minha trajetória, habilidades e projetos. É o ponto central de divulgação do meu portfólio.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura e conteúdo da página
- **CSS3** — Estilização e layout
- **GitHub Pages** — Hospedagem gratuita e publicação automática

## 📁 Estrutura dos Arquivos

```
página-web/
└── README.md       # Documentação desta pasta
index.html          # Arquivo principal da página (na raiz do repositório)
```

## 📌 Versionamento

Este projeto foi versionado com Git, passando pelas seguintes etapas:

- **v1.0** — Criação da estrutura inicial da página
- **v1.1** — Adição de conteúdo e informações pessoais
- **v1.2** — Ajustes de layout e publicação no GitHub Pages
