# 📁 Documentação

Esta pasta centraliza os materiais teóricos, de planejamento e referências utilizados no desenvolvimento deste portfólio profissional.

## 📌 Objetivo

Registrar o processo de criação e organização do repositório, servindo como base de consulta e evidência do planejamento adotado ao longo do projeto.

## 🗂️ O que você encontra aqui

| Conteúdo | Descrição |
|---|---|
| Planejamento do repositório | Definição das pastas, estrutura e objetivos de cada seção |
| Referências e materiais de apoio | Documentos utilizados como base para o desenvolvimento |
| Registro de decisões | Justificativas para as escolhas feitas na organização do portfólio |

## 🛠️ Ferramentas Utilizadas no Projeto

- **Git** — Versionamento local do código
- **GitHub** — Hospedagem e compartilhamento do repositório
- **GitHub Pages** — Publicação da página web estática
- **LinkedIn** — Integração e divulgação profissional

## 📐 Estrutura Geral do Repositório

