# 🎓 Projetos Acadêmicos

Esta pasta reúne os projetos e trabalhos desenvolvidos ao longo da minha graduação em **Ciência de Dados e Machine Learning**.

## 📚 Projetos

### 📊 Análise Exploratória de Dados
**Disciplina:** Fundamentos de Ciência de Dados  
**Descrição:** Projeto de análise exploratória utilizando Python e pandas para identificar padrões em um conjunto de dados real. Inclui limpeza, visualização e interpretação dos dados.  
**Status:** ✅ Concluído

---

### 🤖 Introdução ao Machine Learning
**Disciplina:** Aprendizado de Máquina  
**Descrição:** Implementação de modelos de classificação e regressão com scikit-learn. Foram aplicados algoritmos como KNN, Árvore de Decisão e Regressão Linear.  
**Status:** ✅ Concluído

---

### 🗄️ Banco de Dados Relacional
**Disciplina:** Banco de Dados  
**Descrição:** Modelagem e criação de banco de dados relacional com SQL. Inclui criação de tabelas, relacionamentos, consultas e relatórios.  
**Status:** ✅ Concluído

---

### 🌐 Desenvolvimento Web
**Disciplina:** Desenvolvimento de Interfaces  
**Descrição:** Criação de página web utilizando HTML5 e CSS3 com foco em boas práticas de estrutura e semântica.  
**Status:** ✅ Concluído
